# visual-linear-systems-info

Functions to draw linear systems in a funny way


## Install from PYPI

The homepage in pipy is https://pypi.org/project/visual_linear_systems/

```bash
pip install --upgrade visual_linear_systems
```

Using:

```bash
visual-linear-systems-info
```

## Install from source
Installing `visual-linear-systems-info` program

```bash
git clone https://github.com/trucomanx-pylibs/VisualLinearSystems.git
cd VisualLinearSystems
pip install -r requirements.txt
cd src
python -m build
pip install dist/visual_linear_systems-*.tar.gz
```
Using:

```bash
visual-linear-systems-info
```

## Uninstall

```bash
pip uninstall visual_linear_systems
```
