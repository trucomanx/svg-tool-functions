# visual-linear-systems-info

Functions to draw linear systems in a funny way

## 1. Installing

To install the package from [PyPI](https://pypi.org/project/visual_linear_systems/), follow the instructions below:


```bash
pip install --upgrade visual_linear_systems
```

Execute `which visual-linear-systems-info` to see where it was installed, probably in `/home/USERNAME/.local/bin/visual-linear-systems-info`.

### Using

To start, use the command below:

```bash
visual-linear-systems-info
```
